# Go Tetris

Golang Tetris for console window with optional AI

## Compile

```
go get github.com/MichaelS11/go-tetris
go install github.com/MichaelS11/go-tetris
```

## Play

Then run the binary created, tetris or tetris.exe

## Keys during game

| Key | Action |
| --- | --- |
| &larr; | left move |
| z | left rotate |
| x | right rotate |
| &rarr; | right move |
| &darr; | soft drop |
| &uarr; | hard drop |
| spacebar | hard drop |
| p | pause |
| q | quit |
| i | toggle AI |

## Keys start screen

| Key | Action |
| --- | --- |
| &larr; | previous board |
| &rarr; | next board |
| spacebar | start game |
| q | quit |

## Features include

- AI (use i key to toggle)
- Lock delay
- Next piece
- Ghost piece
- Top scores
- Board choices
