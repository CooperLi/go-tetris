# Go Tetris

Golang Tetris for console window with AI


## Compile

go get github.com/MichaelS11/tetris

go install github.com/MichaelS11/tetris


## Play

Then run the binary created, tetris or tetris.exe


## Keys

| Key | Action |
| --- | --- |
| &larr; | left move |
| z | left rotate |
| x | right rotate |
| &rarr; | right move |
| &darr; | soft drop |
| &uarr; | hard drop |
| spacebar | hard drop |
| p | pause |
| q | quit |


## Features include

- AI (use i key to toggle)
- Lock delay
- Next piece
- Ghost piece
- Top scores
